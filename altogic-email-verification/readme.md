# Create SvelteKit App

1. Run `npm create svelte@latest my-app`
2. Select `Skeleton project`
3. Select `Yes, using TypeScript syntax`
4. Select `Yes`
5. Select `Yes`
6. Select `No`
7. Select `No`
8. Run `cd my-app`
9. Run `npx svelte-add@latest tailwindcss`
10. Run `npm install`
11. Run `npm run dev`
